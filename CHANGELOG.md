# Changelog

All notable changes to BattleChambers are documented here.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-06-28

### Added
- Initial release of BattleChambers
- 10 mini-games:
  - Bow Battle
  - Sword Fight
  - TNT Run
  - Block Party
  - Parkour Challenge
  - Sumo Arena
  - Lava Escape
  - Snowball Fight
  - Fish Challenge
  - Hot Potato
- Arena management system
- Player progression and statistics
- Command system with tab completion
- Event-driven architecture
- SQLite/MySQL support
- Leaderboards
- Configuration system
- Comprehensive logging

### Features
- Full Bukkit/Spigot compatibility (1.20+)
- Asynchronous data loading
- Automatic player data saving
- Customizable games and arenas
- Permission system
- Player statistics tracking
- XP and coin system
